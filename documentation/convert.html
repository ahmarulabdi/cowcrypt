Namespace: convert

convert

This static class offers a grab bag of methods for various types of data conversion operations. WARNING: unless otherwise noted, any method that accepts a string input *must* be ASCII-encoded. (ie. no UTF8 / Unicode multibyte characters values are accepted) Use convert.utf8.encode to encode any non-ASCII string data as needed. Ignore this warning at your own peril.
