Class: EVPKDF

EVPKDF

This is a deprecated passphrase-based key derivation class included to provide compatibility with OpenSSL encryption formats. Normally we'd use PBKDF2 for this sort of thing.

new EVPKDF(data)

Creates a new EVPKDF instance
Parameters:

data Object Initialization options for the class, passed automatically into PBKDF2#initialize. Note this class will ignore your choice of hash algorithm and always use MD5.
