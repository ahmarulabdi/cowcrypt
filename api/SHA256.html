Hashes data into the SHA256 class instance.
Parameters:

data string ASCII-encoded binary string data to hash. Any non-ASCII characters should be first encoded out using convert.utf8.encode
options Object <optional>
 Optional options object
Properties
Name Type Argument Default Description
stream boolean <optional>
 false Whether to use streaming (progressive hashing) mode. In streaming mode, you can repeatedly hash data into the SHA256 object. The hash will not be finalized and returned until you call SHA256#finalize. Streaming mode is useful when you have to hash a huge amount of data and you don't want to store all of it in memory at one time.

 'hex' (binary|hex|words) The return format. Default: hex
